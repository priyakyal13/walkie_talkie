Instructions to run the code:

-There are 3 codes in this folder , one for esp32 wrover , and one for esp32 wroom , upload the code to the respective board. The third code is for the Arduino Mega , upload the code to the Arduino Mega.
-Install the Arduino IDE and the ESP32 board in the IDE.
-View the outputs in the respective serial monitors.
-You can also view the outputs in the serial monitor of the Arduino IDE by connecting the Arduino Mega to the computer via USB.
